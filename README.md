# expense-tracker
A Python-based Expense Tracker that helps users record and manage their daily expenses. The application allows users to add expenses, view all recorded expenses, search expenses by category, calculate total spending, and generate category-wise summaries. Data is stored in a CSV file for persistent storage and managed using the Pandas library.

## Features

- Add expenses
- View expenses
- Calculate total spending
- Search expenses by category
- View category summaries
- Delete expenses
- CSV-based storage

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/expense-tracker.git
```

2. Install dependencies

```bash
pip install pandas tabulate
```

3. Run the application

```bash
python expense-tracker.py
```
